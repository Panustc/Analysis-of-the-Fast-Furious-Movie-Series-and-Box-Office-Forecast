

### 《速度与激情》系列电影：商业版图与艺术特质演变分析报告

本研究通过对《速度与激情》全系列 10 部作品的深度复盘，探讨了该系列如何从早期的“赛车犯罪片”演变为全球性的“超级动作大片”，并利用机器学习算法对未来票房进行了量化预测。

#### 🚀 项目亮点
* **多维数据集成**：整合了电影制作成本、全球大盘走势（Market Size）、社交媒体粉丝量以及专业影评网站（豆瓣/烂番茄）等多源数据。
* **分市场建模策略**：针对中国与海外市场的审美差异，独立构建预测模型，显著提升了预测精度。
* **舆情关联分析**：利用 Python 爬虫获取 B 站数据，定量分析了“家人侠（Family）”热梗与官方宣发节点之间的强关联性。

#### 📊 核心分析维度

##### 1. 艺术基因变迁：从“赛车”到“家人”
通过对电影关键元素的时长占比分析，我们发现：
* **赛车元素衰减**：随着系列发展，纯粹的赛车时间及换挡特写动作占比明显下降。
* **动作戏升级**：爆炸次数、枪支使用时间以及打斗时长在后期作品中占据主导。
* **“Family”核心化**：台词中提到 "Family" 或 "Team" 的次数与电影全球票房呈现出正向波动的关联趋势。

##### 2. 商业表现预测：机器学习模型
本项目对比了多种回归算法，并最终采用集成学习方案：
* **模型选择**：对比线性回归与 **XGBoost**，最终选用 XGBoost 捕捉非线性特征，并通过 **L2 正则化（Ridge Regression）** 抑制过拟合。
* **预测精度**：通过 **Leave-One-Out (LOO)** 交叉验证，将全球总票房预测的平均绝对误差（MAE）控制在 **$117.93M**。
* **未来预测**：模型预测《速度与激情 11》全球合并票房约为 **$1429.08M**。

#### 🛠️ 技术栈
* **数据处理**：Power Query (M 语言), Python (Pandas/NumPy)
* **可视化**：Power BI Desktop (DAX 建模), Matplotlib
* **机器学习**：Scikit-learn, XGBoost
* **数据抓取**：`bilibili-api`

#### 📈 结论摘要
《速度与激情》系列的成功已形成标准商业模式：以高昂的制作与宣发成本投入，换取全球市场大盘下的高份额回报。尽管近期口碑波动，但其形成的“家人侠”文化符号已在二创市场（如 B 站）展现出极强的长尾生命力，为续作提供了稳定的基本盘。
